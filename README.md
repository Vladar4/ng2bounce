Nimgame 2 Bounce
================

A [tutorial](https://vladar4.github.io/nimgame2/tut101_bounce.html) game for the [Nimgame 2](https://github.com/Vladar4/nimgame2) engine.

Controls:
---------

| Action            | Control     |
| ----------------- | ----------- |
| Left paddle up    | Q           |
| Left paddle down  | A           |
| Right paddle up   | Arrow Up    |
| Right paddle down | Arrow Down  |
| Pause             | P           |
